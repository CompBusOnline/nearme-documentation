execute_query

Use to execute defined graph query. Will raise an exception when query return errors.

Arguments:

  • query_name [String]: name of the graphql query defined in marketplace_builder/graphql_queries
  • result_name [String]: name of the variable where results will be strored, default: 'g'

Examples:

{% execute_query "users_query", result_name: "g", current_user: current_user %}
