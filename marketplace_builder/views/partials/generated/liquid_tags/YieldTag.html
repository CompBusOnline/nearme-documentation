{% raw %} ## yield Execute code wrapped inside content_for ### Arguments: - **content_for_symbol** [String]: name of content_for block ### Examples: Use this in a liquid view first. Then you can use yield inside the layout (for example) or another subsequently rendered view, like below: ```liquid {% content_for 'content_for_name' %}Hello world{% endcontent_for %} # another_file.liquid {% yield 'content_for_name' %} ``` {% endraw %}
