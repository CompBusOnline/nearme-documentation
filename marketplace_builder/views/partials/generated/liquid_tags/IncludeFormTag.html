{% raw %} ## include_form This tag should be used to render forms defined in marketplace_builder/form_configurations ### Arguments: - **form_name** [String] name of the form configuration from marketplace_builder/form_configurations - **resource_id** [ID] id of the resource you want to edit - **parent_resource_id** [String] name of the resource type, ex. name of custom_model_type, transactable_type, user_profile_type ### Examples: ```liquid {% include_form 'signup_form' %} {% include_form 'edit_user', resource_id: user.id %} {% include_form 'manager_invite_to_interview', parent_resource_id: 'invitation', user: g.user %} ``` {% endraw %}
