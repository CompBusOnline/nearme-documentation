admin_liquid_layout_delete

Creates liquid layout

Input fields

id (ID!)

Return fields

body (String)

liquid code to evaluate to valid html. To inject content of the resources using the layout use {{ content_for_layout }}

format (PageFormat)

id (ID!)

path (String!)

defines the the path for layout, must starts with layouts/, for example layouts/my_layout

physical_file_path (String)

defines where the file definition will be available after pull; should start with the module name follow by private/public scope, for example manual/public/views/layouts/my_layout
