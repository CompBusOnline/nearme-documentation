admin_liquid_partial_delete

Creates liquid partial

Input fields

id (ID!)

Return fields

body (String)

liquid code to evaluate to valid html

format (PageFormat)

id (ID!)

path (String!)

defines the name of the partial to be used with include tag

physical_file_path (String)

defines where the file definition will be available after pull; should start with the module name follow by private/public scope, for example manual/public/views/partials/my_partial
