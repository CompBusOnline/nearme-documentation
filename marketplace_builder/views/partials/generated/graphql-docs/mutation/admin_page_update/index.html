admin_page_update

id (ID!)

page (UpdatePageInputType!)

Return fields

authorization_policies ([AuthorizationPolicy])

associated authorization policies

cache_for (Int)

the number of seconds for which this page will be cached for not logged in users.

content (String)

format (PageFormat)

handler (PageRequestMethod)

id (ID!)

layout_name (String)

max_deep_level (Int)

max nesting of the url to be resolved by this slug. For example slug "abc" with max_deep_level 2 will resolve /abc, abc/1, abc/2, but will NOT resolve abc/2/something.

metadata (HashObject)

physical_file_path (String)

defines where the file definition will be available after pull; should start with the module name follow by private/public scope, for example manual/public/views/pages/my_page

redirect_code (Int)

redirect_to (String)

request_method (PageRequestMethod)

response_headers (HashObject)

searchable (Boolean)

slug (String!)
